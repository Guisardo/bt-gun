<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Control disconnect/reconnect to UDP trust | Connectionless UDP can outlive trusted control state. |
| Session change to active haptic | Old commands can continue or replay unless explicitly cancelled. |
| Manual smoke to physical validation | Device/LAN behavior needs explicit operator checks after automated tests. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-19 | Tampering | UDP after control disconnect | mitigate | Allow only unchanged-session grace window; reject frames after grace and require fresh auth for new/changed sessions per D-15. |
| T-04-20 | Tampering/Denial of Service | stale input state | mitigate | Clear buttons only on timeout, preserve aim as stale, and prevent old frames from applying per D-13. |
| T-04-21 | Denial of Service | stale haptic | mitigate | TTL, latest-valid-wins, session-change cancel, and no command queue across new sessions per D-08, D-11, and D-14. |
| T-04-22 | Information Disclosure | final docs/status/manual smoke | mitigate | Redaction gate forbids QR secrets, manual codes, proofs, stream keys, and HMAC key material. |
| T-04-23 | Spoofing/Tampering | reconnect session trust | mitigate | New or changed sessions require fresh authenticated control before input is trusted. |
</threat_model>
