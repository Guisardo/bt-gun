## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Smoke command -> native helper | Kotlin smoke launches helper and dispatches input reports. |
| Separate HID output probe -> helper delegate | A macOS HID client sends OS-origin output report bytes. |
| CoreHID gate -> DriverKit fallback | Failed proof can trigger heavier system-extension path. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-17 | Spoofing | HID enumeration | mitigate | Match device by name, VID `0x1209`, PID `0xB707`, report IDs, and sanitized `hidutil`/`ioreg` evidence. |
| T-07-18 | Tampering | Output probe bytes | mitigate | Probe emits exact report ID/version/length/zero flags/reserved bytes; Kotlin mapper validates again before haptic command creation. |
| T-07-19 | Repudiation | OS-origin output proof | mitigate | Require separate `IOHIDDeviceSetReport` client/probe; reject `simulateOutputReport` and direct haptic command as proof. |
| T-07-20 | Information Disclosure | Smoke XML/manifest | mitigate | Store only sanitized case names/status/timing/redacted paths; no fixture hex, QR/manual codes, proof values, stream keys, HMAC keys, private keys, Bluetooth addresses, device ids, or screenshots. |
| T-07-21 | Elevation of Privilege | DriverKit fallback trigger | mitigate | Fallback starts only on recorded CoreHID failure gate per D-02/D-03/D-09. |
| T-07-SC | Tampering | Package installs | accept | No npm, pip, cargo, Homebrew, or new Gradle dependency install in this plan. |
