## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-15 | Information Disclosure | CI signing certificate | mitigate | Private key only in GitHub Actions secret; workflow masks password and never commits PFX. |
| T-06-16 | Tampering | CI artifact | mitigate | Package script records hashes/metadata and uploads one named artifact. |
| T-06-17 | Elevation of Privilege | Install scripts | mitigate | Require explicit user approval before boot/signing/reboot/install/removal actions per D-03. |
| T-06-18 | Information Disclosure | Docs | mitigate | Docs contain no LAN/session secrets, private keys, certificate passwords, QR data, or raw proof values. |
| T-06-SC | Tampering | GitHub Actions supply chain | mitigate | Use first-party GitHub actions, pin action major versions, and record WDK/MSBuild versions in artifact metadata. |
</threat_model>
