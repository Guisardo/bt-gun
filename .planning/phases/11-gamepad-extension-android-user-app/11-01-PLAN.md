## Phase Goal

**As a** player, **I want to** open a minimal `Gamepad Extension` Android app, choose LAN or Bluetooth HID, and play through a camera HUD, **so that** v1 debug tools stay separate from the small user-facing app.

<objective>
Implement the first buildable v1.1 vertical slice:

- GSD v1.1 docs guard.
- `:runtime` split plus debug `:app` preservation.
- New `:user-app` shell with Figma-style HUD, camera gate, landscape support, and zoom rail fallback.
- `PlayModeController` and service output gating.
- Reload recenter-only behavior.
- Profile schema v2 seed for per-profile calibration and soft Back/Home/Select controls.
- Compact v2 UDP and haptic timeline codec/validation tests sufficient to lock contracts before deeper UI wiring.
</objective>
