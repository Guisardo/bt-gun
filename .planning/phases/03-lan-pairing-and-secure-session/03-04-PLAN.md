## Phase Goal

**As a** user with the Android host app and desktop companion, **I want to** pair Android and desktop locally without manual IP entry, **so that** the devices establish a trusted authenticated session for later input streaming.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| WSS client to desktop server | Untrusted network frames enter reliable control server. |
| Pre-auth WebSocket to trusted control state | A socket exists before proof has succeeded. |
| Control diagnostics to UI/logs | Session state is displayed but secrets must stay hidden. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

</threat_model>
