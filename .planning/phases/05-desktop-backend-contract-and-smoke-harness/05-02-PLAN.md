## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Backend capability report -> later diagnostics/planners | Capability fields can overstate real platform support if unsupported reasons are omitted. |
| Stub backend -> future OS adapter | Stub behavior can be mistaken for real OS-visible HID behavior. |
