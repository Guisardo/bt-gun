## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Simulated output report -> haptic command | Future platform output report semantics must route into existing authenticated control, not a new haptic channel. |
| Desktop control channel -> Android phone vibration | Haptic pass requires active paired Android session; no active session must fail closed. |
| Evidence manifest -> committed repo | Human-observed smoke evidence must not leak secrets, device identifiers, or raw logs. |
